CryptoButler

  • Category: Web

  • Tags:
    • / CryptoCurrency
    • / ReactJS
    • / CoinGecko API

CryptoButler is a web app that serves 3rd party data about cryptocurrencies. It allows the user to access the current price, market cap, and volume of whatever cryptocurrency the user may desire.

The frontend of the app is created in ReactJS and Tailwind CSS is used for the graphical interface. The app pulls data for the cryptocurrencies from the CoinGecko API.

The app was initially used to generate ad revenue, but is no longer hosted on its own domain or monetised. A link for the basic version of the app can be found here.
