Digital Painting

This is an installation originally created for a university project, although it has been used on other occasions as well.

The installation gives the user an initial image to paint on which is the first image of an array. By dipping the roller in the "paint" bucket that is also part of the installation, the next image of the array is selected and so on. The paint brush then paints with the next image from said array. This allows the user to paint a sort of collage of different images.

The depth sensor of a Microsoft Kinect is used for tracking where the paint roller is, and via pixel mapping pixels from the selected image are applied as “paint”. The user can dip the roller in a bucket which has a sensor connected to an Arduino board. This action changes the image used as paint. All of these features are handled in an application written in the processing programming language.
