This is a new take on the classic Space Invaders game. This version is a multiplayer game and the visual style is inspired by the suprematism art style.

Suprematism is an abstract art style, characterised by simple geometric shapes, particularly squares and rectangles,.

In the middle of the game screen there is an abstract mothership spawning equally abstract enemies for the players to shoot. Contrary to the original Space Invaders, there is no score. Instead the longest surviving player wins. Players can shoot each other although the enemies often pose a greater threat. When player shots collide with enemies, they split into fragments to increase the intensity of the game.

The game is developed using ActionScript. This means that flash is required to run the game. It can be downloaded from the link below. The player controls are mapped to WASD for player 1 and arrow keys for player 2.
